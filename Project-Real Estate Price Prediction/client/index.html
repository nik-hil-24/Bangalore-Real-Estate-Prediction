Project Description

The main goal of this project is to make a model that can predict the prices of house in Bangaluru City. The data file was preprocessed using various methods like filling the missing values, deleting the redundant features, removing outliers, etc. A model was made using the best regressor, the backend was done using flask and frontend using HTML, CSS, Javascript.
